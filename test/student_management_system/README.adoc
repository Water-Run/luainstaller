= Student management sample

This multi-file CLI is the primary `cjson` packaging fixture. It stores student
records in JSON and supports both interactive use and deterministic commands for
automated tests.

== Dependency

[source,sh]
----
luarocks install lua-cjson
lua -e 'require("cjson"); print("cjson ok")'
----

The native `cjson` module must match the Lua major/minor ABI used for the build
and the target architecture.

== Files

`main.lua` parses commands and runs the interactive menu. `model.lua` validates
records and calculates grades. `service.lua` owns CRUD, search, ranking, and CSV
operations. `storage.lua` performs replacement-safe JSON persistence.
`reports.lua` formats summaries; `utils.lua` provides terminal, CSV, and file
helpers.

== Run from source

[source,sh]
----
data=/tmp/luainstaller-students.json
lua test/student_management_system/main.lua --data "$data" seed
lua test/student_management_system/main.lua --data "$data" list --sort average
lua test/student_management_system/main.lua --data "$data" report
lua test/student_management_system/main.lua --data "$data" rank --course lua
----

Add a record:

[source,sh]
----
lua test/student_management_system/main.lua \
  --data /tmp/luainstaller-students.json add \
  --name 'Ivy Chen' --gender F --class CS2 --birth 2004 \
  --phone 5550109 --email ivy@example.test \
  --grades lua=96,python=91,math=93,english=88
----

CSV operations use `export --out FILE` and `import --file FILE`. Run
`lua test/student_management_system/main.lua --help` for the full command list.

== Package and verify

[source,sh]
----
lua test/student_management_system/smoke_test.lua
luai -t test/student_management_system/main.lua --max-deps 250
luai -b --dir test/student_management_system/main.lua \
  -o build/student-manager --max-deps 250

EMPTY_PATH=$(mktemp -d /tmp/luainstaller-empty-path-XXXXXX)
env -i PATH="$EMPTY_PATH" \
  build/student-manager/student-manager \
  --data /tmp/bundled-students.json seed
rmdir "$EMPTY_PATH"
----

Expected discovery is five project Lua modules plus native `cjson`. The selected
data file is runtime state and is not embedded in the bundle.
