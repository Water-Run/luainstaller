[[scope]]
== Scope

`luainstaller` packages Lua programs into same-environment executables. It is
not a universal cross-compiler. The current compatibility boundary is:

* same OS family;
* same architecture;
* same ABI;
* same Lua ABI;
* compatible native library set.

[IMPORTANT]
====
The documentation describes implemented behavior unless a section explicitly
labels something as a limitation or future direction.
====

[[commands]]
== Command Names

[horizontal]
`luai`:: Compact, script-oriented CLI.
`luainstaller`:: Full-word, human-oriented CLI.

Both command names call the same Lua API underneath. They differ in syntax and
presentation, not in packaging capability.
