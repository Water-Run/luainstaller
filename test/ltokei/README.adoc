= ltokei sample

This source counter is the filesystem-oriented packaging fixture. It walks a
tree with LuaFileSystem, classifies files, and prints code/comment/blank totals
by language.

== Dependency

[source,sh]
----
luarocks install luafilesystem
lua -e 'require("lfs"); print("lfs ok")'
----

== Structure

`main.lua` parses the root path. `src/ltokei/languages.lua` selects language
rules, `scanner.lua` walks and aggregates files, and `formatter.lua` prints the
table. `fixtures/` is the deterministic smoke input.

== Run and package

[source,sh]
----
lua test/ltokei/main.lua test/ltokei/fixtures
lua test/ltokei/smoke_test.lua

luai -t test/ltokei/main.lua --max-deps 120
luai -b --dir test/ltokei/main.lua -o build/ltokei --max-deps 120
EMPTY_PATH=$(mktemp -d /tmp/luainstaller-empty-path-XXXXXX)
env -i PATH="$EMPTY_PATH" build/ltokei/ltokei test/ltokei/fixtures
rmdir "$EMPTY_PATH"
----

Expected discovery is three `ltokei.*` Lua modules plus native `lfs`.
